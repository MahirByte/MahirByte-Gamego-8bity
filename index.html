Gamego 8-Bity

8-bit emulator with BIOS boot, VGA-style font, and NES-like square wave audio.

Power Off
CPU Idle
RAM 0x0000

Controls

  • Arrow Keys — Navigate BIOS / Game Selector
  • Enter — Confirm selection
  • Esc — Return to Game Selector
  • Space — Trigger APU square beep in-game

Sample Base64 ROM

Use this base64 string to load the demo game from another device.
